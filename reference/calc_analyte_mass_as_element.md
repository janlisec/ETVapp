# Calculate the analyte mass as element

Calculate the analyte mass as element.

## Usage

``` r
calc_analyte_mass_as_element(
  R_m,
  K,
  Asp_iso1,
  Asp_iso2,
  As_iso1,
  As_iso2,
  N_sp
)
```

## Arguments

- R_m:

  A vector of isotope ratios.

- K:

  Massbias correction factor.

- Asp_iso1:

  Abundance of the spike isotope in the spike.

- Asp_iso2:

  Abundance of the sample isotope in the spike.

- As_iso1:

  Natural abundance of the spike isotope.

- As_iso2:

  Natural abundance of the sample isotope.

- N_sp:

  Amount of spike.

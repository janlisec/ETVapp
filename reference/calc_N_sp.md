# Calculate the N_sp parameter

Calculate the N_sp parameter.

## Usage

``` r
calc_N_sp(c_sp, V_sp, VF1, M_sp, M_sa)
```

## Arguments

- c_sp:

  Concentration of spike isotope in spike stock solution \[mg/L\].

- V_sp:

  Volume of spike solution \[µL\].

- VF1:

  Dilution factor of spike solution.

- M_sp:

  Molar mass of spike.

- M_sa:

  Molar mass of sample.
